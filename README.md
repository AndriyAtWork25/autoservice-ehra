# 🚗 Autoservice Ehra – Business Website

Modern, responsive website built for a local car service business in Ehra.

## 🌐 Live Demo
👉 https://your-domain.com

---

## 📌 Overview
This project is a business-focused website designed to:
- attract local customers
- build trust
- enable fast contact (phone & WhatsApp)

The main focus was on:
- performance
- clean UI/UX
- conversion (calls & inquiries)

---

## ⚙️ Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS

---

## ✨ Features
- Fully responsive (mobile-first)
- Call-to-action buttons (Call & WhatsApp)
- Modern and clean design
- SEO basics implemented
- Optimized for free hosting (e.g. Vercel)

---

## 📱 Mobile Optimization
The website is optimized for:
- fast loading speed
- easy navigation
- quick contact access

---

## 🧠 Key Learnings
- building for real business needs instead of overengineering
- designing UI/UX for local service businesses
- focusing on conversion and usability
